Before we get started…

Make sure you’re well prepared and have PHP7 and composer as well as a text editor installed on you computer

Installing PHP7

The easiest way to install php…

For windows to download from https://windows.php.net/download

For mac use Homebrew https://brew.sh/ and run `brew install php71`

Installing git

To install git on your computer… go to
http://www.git-scm.com/downloads… and download the installer for your system and run it

Installing composer

To install the PHP package manager composer on your computer… go to
https://getcomposer.org/… and follow the instructions.

Check if it all works

To check if everything works, open a console and type:

$ php -v
PHP 7.1.14 (cli) (built: Feb  2 2018 08:42:59) ( NTS
$ composer --version
Composer version 1.6.3 2018-01-31 16:28:17
$ git --version
git version 2.16.2

If you’re having problems, ask one of the mentors to take a look and help sort it out

Installing a text editor

Any text editor will do, but we are using Atom, Sublime or PHPStorm.

Got everything installed?

Awesome! Get yourself an account on GitHub and join the developer community!

This is optional, but will allow you to get more out of PHPSchool

Signing up for GitHub

Go to https://github.com/ and fill out your details…, verify your account by following the instructions in the email… and presto, you’re done!

You’re ready for PHPSchool to begin!
